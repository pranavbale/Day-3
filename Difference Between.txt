Window:
The Window interface is home to a variety of functions, namespaces, objects, and constructors which are not 
necessarily directly associated with the concept of a user interface window. However, the Window interface 
is a suitable place to include these items that need to be globally available.

Screen: 
The JavaScript screen object holds information of browser screen. It can be used to display screen width, 
height, colorDepth, pixelDepth etc.

Document:
The document object represents your web page. If you want to access any element in an HTML page, you always 
start with accessing the document object. it is used to functalize the web page.
